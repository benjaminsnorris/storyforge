## Embeddable block

The canonical motif description. Cover: visual specifics (what does it
look like exactly), placement rules (where in the panel does it sit),
how it reads at a glance, what it pairs with.

## Clauses

One bullet per element. For each, state the failure mode it counters.
Motifs are especially vulnerable to model drift — explicit clauses
prevent the motif from morphing across appearances.

## Iteration history

Date, what changed, why. Track how the motif evolved — when it shifted
in meaning, when its visual signature tightened, what cross-scene rhymes
were added.
