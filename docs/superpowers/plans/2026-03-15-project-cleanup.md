# Project Cleanup Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a cleanup script and interactive skill that brings novel projects in line with the current plugin version — fixing structural drift, pruning working directory bloat, and reporting integrity issues for interactive resolution.

**Architecture:** A `storyforge-cleanup` bash script handles all deterministic fixes (gitignore, missing dirs, yaml migration, CSV columns, junk cleanup, legacy deletion, review dedup). It outputs structured report lines for integrity issues it can't auto-fix. A `/storyforge:cleanup` skill runs the script, presents findings, and handles judgment-call items interactively (unexpected files, character drift, orphan scenes, CLAUDE.md regeneration).

**Tech Stack:** Bash (set -eo pipefail, BSD sed compat), pipe-delimited CSV, common.sh shared libraries, Claude Code skill (SKILL.md).

**Spec:** `docs/superpowers/specs/2026-03-15-project-cleanup-design.md`

Create `scripts/storyforge-cleanup`:

```bash
#!/bin/bash
set -eo pipefail

# ============================================================================
# storyforge-cleanup — Project structure cleanup and migration
#
# Fixes structural drift in Storyforge novel projects: updates gitignore,
# creates missing directories, migrates storyforge.yaml, adds CSV columns,
# removes junk files, deletes legacy artifacts, and reports integrity issues.
#
# Usage:
#   ./storyforge cleanup                  # Apply all fixes and commit
#   ./storyforge cleanup --dry-run        # Report what would change
#   ./storyforge cleanup --verbose        # Detailed output
# ============================================================================

# --- Allow sourcing for tests ---
if [[ "${1:-}" == "--source-only" ]]; then
    # Just define functions, don't execute
    _CLEANUP_SOURCE_ONLY=true
    shift
fi

# --- Locate script directory and source libraries ---
SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"
LIB_DIR="${SCRIPT_DIR}/lib"

if [[ ! -f "${LIB_DIR}/common.sh" ]]; then
    echo "ERROR: Cannot find lib/common.sh at ${LIB_DIR}" >&2
    exit 1
fi
source "${LIB_DIR}/common.sh"

# ============================================================================
# Gitignore
# ============================================================================

# Required gitignore entries — each is checked individually
GITIGNORE_ENTRIES=(
    "# Logs (debugging output, value extracted at write time)"
    "working/logs/"
    ""
    "# Batch API payloads (keep only latest for debugging)"
    "working/scores/**/.batch-requests.jsonl"
    ""
    "# Intermediate scoring/eval state"
    "working/evaluations/**/.status-*"
    "working/scores/**/.markers-*"
    ""
    "# Temporary flag files (cleaned up by scripts)"
    "working/.autopilot"
    "working/.interactive"
)

# Functional entries that must be present (not comments or blank lines)
GITIGNORE_REQUIRED=(
    "working/logs/"
    "working/scores/**/.batch-requests.jsonl"
    "working/evaluations/**/.status-*"
    "working/scores/**/.markers-*"
    ".DS_Store"
    "working/.autopilot"
    "working/.interactive"
)

update_gitignore() {
    local project_dir="$1"
    local gitignore="${project_dir}/.gitignore"

    # Create if missing
    if [[ ! -f "$gitignore" ]]; then
        cat > "$gitignore" << 'GIEOF'
# Storyforge — Novel Project .gitignore

# macOS
.DS_Store

GIEOF
    fi

    local content
    content=$(cat "$gitignore")
    local added=false

    for entry in "${GITIGNORE_REQUIRED[@]}"; do
        if ! grep -qF "$entry" "$gitignore"; then
            added=true
        fi
    done

    if $added; then
        # Rebuild: keep existing content, append missing sections
        local new_content="$content"

        # Ensure trailing newline
        [[ "${new_content: -1}" != $'\n' ]] && new_content="${new_content}"$'\n'

        # Check and append each required entry with its comment
        if ! grep -qF "working/logs/" "$gitignore"; then
            new_content+=$'\n'"# Logs (debugging output, value extracted at write time)"$'\n'
            new_content+="working/logs/"$'\n'
        fi

        if ! grep -qF "working/scores/**/.batch-requests.jsonl" "$gitignore"; then
            new_content+=$'\n'"# Batch API payloads (keep only latest for debugging)"$'\n'
            new_content+="working/scores/**/.batch-requests.jsonl"$'\n'
        fi

        if ! grep -qF "working/evaluations/**/.status-*" "$gitignore"; then
            new_content+=$'\n'"# Intermediate scoring/eval state"$'\n'
            new_content+="working/evaluations/**/.status-*"$'\n'
        fi

        if ! grep -qF "working/scores/**/.markers-*" "$gitignore"; then
            new_content+="working/scores/**/.markers-*"$'\n'
        fi

        if ! grep -qF "working/.interactive" "$gitignore"; then
            # Check if autopilot section exists; if so, add interactive there
            if grep -qF "working/.autopilot" "$gitignore" && ! grep -qF "working/.interactive" "$gitignore"; then
                new_content=$(echo "$new_content" | sed '/working\/.autopilot/a\
working/.interactive')
            else
                new_content+=$'\n'"# Temporary flag files (cleaned up by scripts)"$'\n'
                new_content+="working/.autopilot"$'\n'
                new_content+="working/.interactive"$'\n'
            fi
        fi

        echo "$new_content" > "$gitignore"
    fi
}

# ============================================================================
# Main (skip if sourced for testing)
# ============================================================================

if [[ "${_CLEANUP_SOURCE_ONLY:-}" == "true" ]]; then
    return 0 2>/dev/null || true
fi
```

# ============================================================================
# Junk File Cleanup
# ============================================================================

clean_junk_files() {
    local project_dir="$1"
    local count=0

    # .status-* files in evaluation dirs
    find "${project_dir}/working/evaluations" -name ".status-*" -delete 2>/dev/null

    # .markers-* files in score dirs
    find "${project_dir}/working/scores" -name ".markers-*" -delete 2>/dev/null

    # .batch-requests.jsonl in score dirs EXCEPT latest/
    find "${project_dir}/working/scores" -name ".batch-requests.jsonl" -not -path "*/latest/*" -delete 2>/dev/null

    # All log files
    if [[ -d "${project_dir}/working/logs" ]]; then
        find "${project_dir}/working/logs" -type f -delete 2>/dev/null
    fi

    # Empty working subdirectories
    for dir in enrich coaching backups scenes-setup; do
        local target="${project_dir}/working/${dir}"
        if [[ -d "$target" ]] && [[ -z "$(ls -A "$target" 2>/dev/null)" ]]; then
            rmdir "$target"
        fi
    done
}

# ============================================================================
# storyforge.yaml Migration
# ============================================================================

migrate_storyforge_yaml() {
    local project_dir="$1"
    local yaml="${project_dir}/storyforge.yaml"

    [[ -f "$yaml" ]] || return 0

    local content
    content=$(cat "$yaml")
    local modified=false

    # --- Move misplaced chapter_map to under artifacts ---
    if echo "$content" | grep -q '^chapter_map:'; then
        # Extract the chapter_map block (top-level, with its indented children)
        local cm_block
        cm_block=$(echo "$content" | sed -n '/^chapter_map:/,/^[^ ]/{ /^chapter_map:/p; /^  /p; }')

        # Remove the top-level block
        content=$(echo "$content" | sed '/^chapter_map:/,/^[^ ]/{/^chapter_map:/d; /^  exists:/d; /^  path:/d; /^  updated:/d;}')
        # Clean any leftover blank lines
        content=$(echo "$content" | sed '/^$/{ N; /^\n$/d; }')

        # Insert under artifacts (indent by 2 more spaces)
        local indented_cm="  chapter_map:"$'\n'
        indented_cm+="    exists: $(echo "$cm_block" | grep 'exists:' | sed 's/.*exists: *//')"$'\n'
        indented_cm+="    path: $(echo "$cm_block" | grep 'path:' | sed 's/.*path: *//')"$'\n'
        indented_cm+="    updated: $(echo "$cm_block" | grep 'updated:' | sed 's/.*updated: *//')"

        # Append after last artifact entry (before next top-level key)
        content=$(echo "$content" | sed "/^artifacts:/,/^[^ ]/ {
            /^[^ ]/ {
                /^artifacts:/! i\\
${indented_cm}
            }
        }")

        modified=true
    fi

    # --- Add missing sections ---
    if ! echo "$content" | grep -q '^scene_extensions:'; then
        content+=$'\n'"scene_extensions: []"$'\n'
        modified=true
    fi

    if ! echo "$content" | grep -q '^evaluation:'; then
        content+=$'\n'"evaluation:"$'\n'"  custom_evaluators: []"$'\n'
        modified=true
    fi

    if ! echo "$content" | grep -q '^production:' && ! echo "$content" | grep -q '^# production:'; then
        content+=$'\n'"# production:"$'\n'"#   author: \"\""$'\n'"#   language: en"$'\n'"#   scene_break: ornamental"$'\n'"#   default_heading: numbered-titled"$'\n'
        modified=true
    fi

    if ! echo "$content" | grep -q '^parts:' && ! echo "$content" | grep -q '^# parts:'; then
        content+=$'\n'"# parts:"$'\n'"#   - number: 1"$'\n'"#     title: \"Part One\""$'\n'
        modified=true
    fi

    # --- Artifact flag sync ---
    # Known artifact file mappings (id -> relative path)
    local artifact_files
    artifact_files=(
        "characters:reference/characters.csv"
        "locations:reference/locations.csv"
        "threads:reference/threads.csv"
        "motif_taxonomy:reference/motif-taxonomy.csv"
        "scene_intent:reference/scene-intent.csv"
        "scene_log:reference/scene-log.md"
        "resolved_threads:reference/resolved-threads.md"
        "title_development:reference/title-development.md"
    )

    for entry in "${artifact_files[@]}"; do
        local aid="${entry%%:*}"
        local apath="${entry#*:}"

        if [[ -f "${project_dir}/${apath}" ]] && ! echo "$content" | grep -q "  ${aid}:"; then
            # Add artifact entry under artifacts section
            local new_entry="  ${aid}:"$'\n'"    exists: true"$'\n'"    path: ${apath}"$'\n'"    updated:"

            # Insert before the line after the last artifact
            content=$(echo "$content" | sed "/^artifacts:/,/^[^ ]/ {
                /^[^ ]/ {
                    /^artifacts:/! i\\
${new_entry}
                }
            }")
            modified=true
        fi
    done

    # Fix exists flags for all artifacts using awk for block-level awareness
    # YAML artifact blocks have: name: / exists: / path: / updated:
    # exists: comes BEFORE path: in each block, so we must read the whole block first
    content=$(echo "$content" | awk -v proj="$project_dir" '
    /^  [a-z_]+:$/ || /^  [a-z_]+: *$/ {
        # Start of an artifact block (2-space indented key with no value)
        block_name = $0
        block_lines[0] = $0
        block_count = 1
        in_block = 1
        next
    }
    in_block && /^    exists:/ {
        block_lines[block_count++] = $0
        next
    }
    in_block && /^    path:/ {
        block_lines[block_count++] = $0
        # Extract path value
        sub(/^    path: */, "", $0)
        gsub(/"/, "", $0)
        gsub(/ *$/, "", $0)
        block_path = $0
        next
    }
    in_block && /^    updated:/ {
        block_lines[block_count++] = $0
        # Now we have the full block — check exists vs disk
        cmd = "test -e \"" proj "/" block_path "\" && echo yes || echo no"
        cmd | getline disk_exists
        close(cmd)
        for (i = 0; i < block_count; i++) {
            line = block_lines[i]
            if (line ~ /exists: false/ && disk_exists == "yes") {
                sub(/exists: false/, "exists: true", line)
            } else if (line ~ /exists: true/ && disk_exists == "no") {
                sub(/exists: true/, "exists: false", line)
            }
            print line
        }
        in_block = 0
        block_count = 0
        block_path = ""
        next
    }
    in_block && !/^    / {
        # Left the block without seeing all fields — flush as-is
        for (i = 0; i < block_count; i++) print block_lines[i]
        in_block = 0
        block_count = 0
        print
        next
    }
    in_block {
        block_lines[block_count++] = $0
        next
    }
    { print }
    ')
    modified=true

    if $modified; then
        echo "$content" > "$yaml"
    fi
}

# CSV Integrity Report

report_csv_integrity() {
    local project_dir="$1"
    local meta_csv="${project_dir}/reference/scene-metadata.csv"
    local intent_csv="${project_dir}/reference/scene-intent.csv"
    local chapter_csv="${project_dir}/reference/chapter-map.csv"
    local chars_csv="${project_dir}/reference/characters.csv"
    local scenes_dir="${project_dir}/scenes"

    # --- Scene files vs metadata ---
    if [[ -f "$meta_csv" ]] && [[ -d "$scenes_dir" ]]; then
        local meta_ids
        meta_ids=$(tail -n +2 "$meta_csv" | cut -d'|' -f1 | sort)

        local file_ids
        file_ids=$(ls "${scenes_dir}"/*.md 2>/dev/null | xargs -I{} basename {} .md | sort)

        # Files without metadata
        comm -23 <(echo "$file_ids") <(echo "$meta_ids") | while IFS= read -r id; do
            [[ -n "$id" ]] && echo "ORPHAN_FILE:${id}"
        done

        # Metadata without files
        comm -13 <(echo "$file_ids") <(echo "$meta_ids") | while IFS= read -r id; do
            [[ -n "$id" ]] && echo "ORPHAN_META:${id}"
        done
    fi

    # --- Metadata vs intent ---
    if [[ -f "$meta_csv" ]] && [[ -f "$intent_csv" ]]; then
        local meta_ids
        meta_ids=$(tail -n +2 "$meta_csv" | cut -d'|' -f1 | sort)
        local intent_ids
        intent_ids=$(tail -n +2 "$intent_csv" | cut -d'|' -f1 | sort)

        comm -23 <(echo "$meta_ids") <(echo "$intent_ids") | while IFS= read -r id; do
            [[ -n "$id" ]] && echo "MISSING_INTENT:${id}"
        done

        comm -13 <(echo "$meta_ids") <(echo "$intent_ids") | while IFS= read -r id; do
            [[ -n "$id" ]] && echo "EXTRA_INTENT:${id}"
        done
    fi

    # --- Chapter map references ---
    if [[ -f "$chapter_csv" ]] && [[ -f "$meta_csv" ]]; then
        local meta_ids
        meta_ids=$(tail -n +2 "$meta_csv" | cut -d'|' -f1 | sort)

        tail -n +2 "$chapter_csv" | cut -d'|' -f5 | tr ';' '\n' | sed 's/^ *//;s/ *$//' | grep -v '^$' | sort -u | while IFS= read -r id; do
            if ! echo "$meta_ids" | grep -qx "$id"; then
                echo "BAD_CHAPTER_REF:${id}"
            fi
        done
    fi

    # --- Sequence gaps ---
    if [[ -f "$meta_csv" ]]; then
        local prev=0
        tail -n +2 "$meta_csv" | cut -d'|' -f2 | sort -n | while IFS= read -r seq; do

            cd "$PROJECT_DIR" && echo "$ignored_tracked" | xargs git rm --cached -q 2>/dev/null || true
        fi
    fi
fi

# Step 2: Missing directories
log "Checking directories..."
if $DRY_RUN; then
    for dir in "${EXPECTED_DIRS[@]}"; do
        [[ ! -d "${PROJECT_DIR}/${dir}" ]] && log "  Would create ${dir}/"
    done
else
    CREATED=$(create_missing_dirs "$PROJECT_DIR")
    [[ -n "$CREATED" ]] && echo "$CREATED" | while IFS= read -r d; do
        [[ -n "$d" ]] && vlog "  Created ${d}/"
    done
fi

# Step 3: storyforge.yaml migration
log "Checking storyforge.yaml..."
if $DRY_RUN; then
    DRY_TMP="${TMPDIR}/cleanup-dry-yaml-$$"
    mkdir -p "$DRY_TMP/reference"
    cp "${PROJECT_DIR}/storyforge.yaml" "$DRY_TMP/storyforge.yaml" 2>/dev/null
    # Copy reference files so artifact flag sync can check disk
    cp -R "${PROJECT_DIR}/reference/" "$DRY_TMP/reference/" 2>/dev/null || true
    migrate_storyforge_yaml "$DRY_TMP"
    if ! diff -q "${PROJECT_DIR}/storyforge.yaml" "$DRY_TMP/storyforge.yaml" > /dev/null 2>&1; then
        log "  Would migrate storyforge.yaml (missing sections, artifact flags)"
    fi
    rm -rf "$DRY_TMP"
else
    migrate_storyforge_yaml "$PROJECT_DIR"
fi

# Step 4: Pipeline CSV
log "Checking pipeline.csv..."
if $DRY_RUN; then
    HEADER=$(head -1 "${PROJECT_DIR}/working/pipeline.csv" 2>/dev/null || echo "")
    [[ "$HEADER" != "$PIPELINE_EXPECTED" ]] && log "  Would add missing columns to pipeline.csv"
else
    migrate_pipeline_csv "$PROJECT_DIR"
fi

# Step 5: Junk files
log "Cleaning junk files..."
if $DRY_RUN; then
    COUNT=$(find "${PROJECT_DIR}/working/evaluations" -name ".status-*" 2>/dev/null | wc -l | tr -d ' ')
    (( COUNT > 0 )) && log "  Would remove $COUNT .status-* files"
    COUNT=$(find "${PROJECT_DIR}/working/scores" -name ".markers-*" 2>/dev/null | wc -l | tr -d ' ')
    (( COUNT > 0 )) && log "  Would remove $COUNT .markers-* files"
    COUNT=$(find "${PROJECT_DIR}/working/scores" -name ".batch-requests.jsonl" -not -path "*/latest/*" 2>/dev/null | wc -l | tr -d ' ')
    (( COUNT > 0 )) && log "  Would remove $COUNT .batch-requests.jsonl files"
    COUNT=$(find "${PROJECT_DIR}/working/logs" -type f 2>/dev/null | wc -l | tr -d ' ')
    (( COUNT > 0 )) && log "  Would remove $COUNT log files"
else
    clean_junk_files "$PROJECT_DIR"
fi

# Step 6: Legacy files
log "Checking legacy files..."
if $DRY_RUN; then
    for f in "working/pipeline.yaml" "working/assemble.py"; do
        [[ -f "${PROJECT_DIR}/${f}" ]] && log "  Would delete ${f}"
    done
else
    delete_legacy_files "$PROJECT_DIR"
fi

# Step 7: Reorganize loose files
log "Reorganizing loose files..."
if $DRY_RUN; then
    COUNT=$(ls "${PROJECT_DIR}"/working/recommendations*.md 2>/dev/null | wc -l | tr -d ' ')
    (( COUNT > 0 )) && log "  Would move $COUNT recommendation files to working/recommendations/"
else
    reorganize_loose_files "$PROJECT_DIR"
fi

# Step 8: Pipeline review dedup
log "Deduplicating pipeline reviews..."
if $DRY_RUN; then
    # Count how many would be removed
    log "  Would deduplicate pipeline reviews (keep latest per day)"
else
    dedup_pipeline_reviews "$PROJECT_DIR"
fi

# Step 9: CSV integrity report
log ""
log "=== CSV Integrity Report ==="
report_csv_integrity "$PROJECT_DIR"

# Step 10: Unexpected files report
log ""
log "=== Unexpected Files Report ==="
report_unexpected_files "$PROJECT_DIR"

# Step 11: Commit (unless dry-run)
if ! $DRY_RUN; then
    if command -v git > /dev/null 2>&1 && [[ -d "${PROJECT_DIR}/.git" ]]; then
        cd "$PROJECT_DIR"
        if [[ -n "$(git status --porcelain 2>/dev/null)" ]]; then
            log ""
            log "Committing changes..."
            git add -A
            git commit -m "Cleanup: project structure and working files"
            git push 2>/dev/null || log "WARNING: git push failed (no remote or auth issue)"
        else
            log "No changes to commit."
        fi
    fi
fi

log ""
log "Cleanup complete."
```

# Storyforge Cleanup Skill

You are helping an author clean up and migrate their Storyforge novel project. This skill handles structural drift, working directory bloat, config migration, and data integrity issues.

## Locating the Storyforge Plugin

The Storyforge plugin root is two levels up from this skill file's directory
(this skill's directory → `skills/` → plugin root). Scripts live at `scripts/`
relative to that plugin root.

Store this resolved plugin path for use throughout the session.

## Step 1: Read Project State

Read the following files to understand the current project:

- `storyforge.yaml` — project configuration
- `reference/scene-metadata.csv` — scene metadata
- `reference/scene-intent.csv` — scene intent data (if it exists)
- `reference/characters.csv` — character reference (if it exists)
- `reference/chapter-map.csv` — chapter mapping (if it exists)
- `working/pipeline.csv` — pipeline state (if it exists)
- `.gitignore` — current gitignore rules (if it exists)
- `CLAUDE.md` — project orientation file

Also scan:
- `ls scenes/` — scene files on disk
- `ls working/` — working directory contents
- `ls -d */` — top-level directories

## Step 2: Run the Cleanup Script in Dry-Run Mode

> I'll run the cleanup script in this conversation to see what needs fixing.

```

## Step 3: Present Findings

Summarize the script's dry-run output in two groups:

### Auto-fixable (the script handles these)
- Gitignore updates
- Missing directories
- storyforge.yaml migration
- Pipeline CSV columns
- Junk file cleanup (log files, status files, markers, batch payloads)
- Legacy file deletion
- Loose file reorganization
- Pipeline review deduplication

Present the count and nature of each fix. Ask the author for a go-ahead:

> "The script will [summary of changes]. Want me to run it?"

### Needs Your Input
Parse the report lines from the script output and present these interactively:

**Unexpected directories/files** (`UNEXPECTED_DIR:`, `UNEXPECTED_FILE:`):
For each one, ask: "I found `[path]` which isn't part of the standard Storyforge structure. Keep it or remove it?"

**Orphan scenes** (`ORPHAN_FILE:`, `ORPHAN_META:`):
- `ORPHAN_FILE`: "Scene file `[id].md` exists but has no metadata row. Add a row to scene-metadata.csv, or delete the file?"
- `ORPHAN_META`: "Metadata row for `[id]` exists but there's no scene file. Remove the row, or create a placeholder scene file?"

**Character drift** (`UNKNOWN_CHARACTER:`):
Present the full list of characters found in scenes but not in characters.csv. Ask: "These characters appear in your scenes but aren't in characters.csv. Want me to add entries for them?"

**Missing intent** (`MISSING_INTENT:`, `EXTRA_INTENT:`):
- `MISSING_INTENT`: "These scenes have metadata but no intent row: [list]. Add placeholder intent rows?"
- `EXTRA_INTENT`: "These intent rows have no matching metadata: [list]. Remove them?"

**Bad chapter references** (`BAD_CHAPTER_REF:`):
"Chapter map references these scenes that don't exist: [list]. These need manual attention in the chapter map."

**Sequence gaps** (`SEQ_GAP:`):
"Sequence gap found: [range]. This may be intentional (cut scenes) or accidental. Want me to renumber to close the gaps?"
