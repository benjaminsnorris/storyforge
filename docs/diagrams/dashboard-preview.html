Storyforge

Manuscript Dashboard

Ten visualizations that let you see your entire book

storyforge visualize

Run ./storyforge visualize to generate a self-contained HTML dashboard at working/dashboard.html. All data comes from your scene CSV files. Every view is linked — click a scene in any visualization and it highlights everywhere.

Visualizations

The Ten Views

Each view reveals a different dimension of your manuscript. Together they give you a complete structural picture.

1

Manuscript Spine

A horizontal strip showing every scene as a colored block. Width encodes word count, color encodes scene type. Part dividers mark structural breaks.

2

POV River

Horizontal swimlanes for each point-of-view character. Colored blocks show which POV owns each scene, revealing balance and rhythm.

3

Character Presence Grid

A character-by-scene matrix. Filled dots show where each character appears, revealing co-occurrence patterns and character arcs.

4

Thread Weave

Horizontal rows for each narrative thread. Dots mark scenes where a thread appears, connected by lines. Dashed gaps show thread dormancy.

5

Emotional Terrain

A smooth line chart showing emotional intensity across scenes. Peaks mark high-tension moments; valleys show quiet passages.

6

Scene Type Sequence

A thin colored strip showing scene types in sequence. Reveals the rhythm of action, dialogue, introspection, and transition.

7

Setting Map

Dot strips showing where each setting appears across the manuscript. Reveals geographic rhythm and the weight given to each location.

8

Craft Score Heatmap

A grid of colored cells showing craft dimension scores per scene. Red cells need work, amber is passable, green is strong.

9

Motif Constellation

Like thread weave but for recurring images and motifs. Dots mark appearances, with a density bar below showing concentration.

10

Timeline vs Reading Order

A scatter plot comparing story chronology to reading order. Points near the diagonal mean linear storytelling; deviations reveal flashbacks and time jumps.

Interaction

Linked Selection

All ten views are wired together. Interact with one and the rest respond.

Cross-View Highlighting

Hover over any scene in any visualization and it highlights in all ten views simultaneously. Click to lock the selection. This lets you trace a single scene across every structural dimension — its POV, characters, threads, emotional intensity, craft scores, and timeline position — all at once.

Use the filter controls to narrow by character, thread, or motif. The entire dashboard updates in real time.

Usage

Generation

Generate your dashboard with a single command.

terminal
./storyforge visualize # Generate dashboard
./storyforge visualize --open # Generate and open in browser
