/storyforge:produce interactive ./storyforge assemble autonomous
Section 1

The Production Flow

Four stages transform a collection of scenes into a finished book.

1
Chapter Mapping
/storyforge:produce
Interactive
  • Group scenes into chapters
  • Set chapter titles and heading format
  • Choose scene break style

This is a creative decision — affected by coaching level.

2
Production Settings
/storyforge:produce
Interactive
  • Author name, copyright, ISBN
  • Genre typography preset (6 presets available)
  • Front matter: dedication, epigraph, author's note
  • Back matter: acknowledgments, about the author, also by
  • Cover image
3
Assembly
./storyforge assemble
Autonomous
  • Reads chapter map and scene files
  • Concatenates scenes into chapters
  • Applies typography and formatting
  • Generates output in requested format
4
Output Formats
Generated artifacts
EPUB
EPUB3, validated with epubcheck
PDF
Requires LaTeX or weasyprint
HTML
Single-file, self-contained
Markdown
Assembled markdown, no dependencies
Section 2

The Review Integration

An automatic quality gate at the end of every autonomous process.

Every autonomous process (write, evaluate, revise, assemble) ends with an automatic review phase. This ensures quality at each stage before work continues.
Review Phase Steps
1
Changes PR label from in-progress to reviewing
2
Runs Claude review on the diff
3
Posts review comments on the PR
4
Changes label to ready-to-merge
Integration Flow
write → review → evaluate → review → revise → review → assemble → review
/storyforge:review also works standalone for manual review at any time.
Section 3

The Chapter Map Format

A YAML file that defines how scenes are grouped into chapters and sets production metadata.

chapter-map.yaml 
chapters:
  - title: "The Finest Cartographer"
    heading: numbered-titled
    scenes:
      - act1-sc01
      - act1-sc02

  - title: "Into the Unmarked Territory"
    heading: numbered-titled
    scenes:
      - act1-sc03
      - act1-sc04
      - act1-sc05

production:
  author: "Author Name"
  genre_preset: fantasy
  scene_break: ornamental        # ornamental | simple | space

front_matter:
  dedication: "For the mapmakers."
  epigraph: "Not all who wander..."

back_matter:
  acknowledgments: true
  about_author: true
Section 4

Genre Presets

Six CSS typography presets that control how the finished book looks and feels.

Default
default
Clean serif typography with balanced margins. A versatile baseline that works for any genre.
serif body 1.5 line-height standard margins
Literary Fiction
literary-fiction
Elegant presentation with generous whitespace and drop caps. Designed for prose that breathes.
drop caps wide margins generous leading
Thriller
thriller
Tight typography with bold, sans-serif headings. Short chapters and punchy scene breaks drive momentum.
sans-serif headings tight spacing bold breaks
Romance
romance
Warm Garamond-style typography with decorative scene breaks. Inviting and intimate on the page.
warm serif decorative breaks soft headings
Fantasy
fantasy
Ornamental styling with bold small-cap headings and drop caps. Evokes a sense of world and weight.
small-cap headings drop caps ornamental breaks
Science Fiction
science-fiction
Modern sans-serif with geometric accents. Clean, forward-looking typography for speculative worlds.
sans-serif body geometric accents modern layout
