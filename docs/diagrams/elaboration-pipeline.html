Key Principles

The ideas that make elaboration work.

Validate Cheap, Fix Cheap

Problems caught as CSV edits cost nothing. Problems caught in prose cost revision cycles. Every stage runs validation before advancing.

Parallel Drafting

Scenes with no continuity dependencies draft simultaneously. The dependency graph determines wave grouping — most novels need 3–5 waves for 40–60 scenes.

Evaluation Feeds Upstream

When evaluators find a problem, the fix goes to the CSV that owns it — not to the prose. Revised briefs produce revised scenes with the fix built in.

Coaching Levels Are Roles

Full = creative partner. Coach = dramaturg. Strict = continuity editor. Each role is genuinely valuable — the validation engine alone is worth the price of admission.

Three Ways to Work

Coaching levels determine Claude's role — not how much help, but what kind.

Full

Creative Partner

Claude proposes the spine, builds architecture, writes briefs, drafts scenes. Author reviews at PR gates. Could be surprised by creative choices — the guardrails ensure quality.

Coach

Dramaturg

Claude presents options and asks questions. "Here are three possible spines — which resonates?" Author makes every creative decision; Claude structures and organizes.

Strict

Continuity Editor

Author writes everything. Claude runs validation — exhaustive structural checking across 50+ scenes that's genuinely hard for humans. The validation engine is the product.

Built on Craft

Every column in the data model maps to an established craft principle.

Robert McKee

Value at stake, value shift, polarity tracking. A scene that doesn't shift a value is a nonevent.

Dwight Swain

Action/sequel pattern. Goal → conflict → outcome. Reaction → dilemma → decision. The rhythm of breathing.

Shawn Coyne

Five Commandments per scene. Inciting incident, progressive complications, crisis, climax, resolution.

Mary Robinette Kowal

MICE quotient. Thread typing (milieu, inquiry, character, event) with FILO nesting order.
