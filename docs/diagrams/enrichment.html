Architecture

Two-Phase Enrichment

Extract what you can for free, then call Claude only for the gaps.

Manuscript Scenes
Raw scene files from scenes/ and the scene index YAML. Each scene may be missing some or all enrichment fields.
Phase 1: Free (No API Cost)
Pure shell heuristics that run instantly with zero API calls. Word count via wc -w. Time of day inferred by grepping for keywords like "morning", "2:47 AM", "midnight". Threads recovered from scene-index.yaml (threads_planted / threads_advanced / threads_resolved merged into a single threads field).
word_count time_of_day threads
Phase 2: Claude (Targeted)
Only runs for scenes still missing data after Phase 1. Builds a per-scene prompt asking only for the missing fields — not all seven. Runs parallel workers (default 6) for speed.
type characters emotional_arc motifs timeline_day

Efficiency

Cost Efficiency

Every design choice minimizes API spend without sacrificing completeness.

Phase 1 Is Free

Word counts, time-of-day keywords, and thread recovery all use shell commands with zero API calls. For many fields this is all you need.

Targeted Prompts

Phase 2 only sends scenes that still have gaps. Each prompt asks only for the missing fields, not all seven — smaller prompts, lower cost.

Parallel Workers

Default 6 parallel workers process scenes concurrently. Same total cost, but finishes much faster for large manuscripts.

Cost Ledger

Every API call is tracked in working/costs/ledger.csv so you always know what enrichment costs.

Usage

Commands

Run enrichment from the project root with flexible targeting.

terminal 
./storyforge enrich                    # All scenes, all fields
./storyforge enrich --fields type,characters  # Specific fields only
./storyforge enrich --act 2            # One act only
./storyforge enrich --dry-run          # Preview what would happen

Enrichment is idempotent — running it again skips fields that already have data. Use --fields to target specific columns, or --act to limit scope. Phase 1 always runs first so you never pay for data you could get for free.
