Core Concepts
The Elaboration Pipeline Build structural integrity before writing prose. Spine → architecture → voice → map → briefs → draft → polish. Extraction: Reverse Elaboration Take an existing manuscript and extract structural data. Four phases from prose to validated scene model. Scoring & Fidelity 25 craft principles per scene. Brief fidelity checks whether prose delivers what briefs promised.
